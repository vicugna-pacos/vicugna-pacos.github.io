概要

QnAMaker で FAQ チャットボットを運用するときに、例えば経理部 FAQ の場合だと経理部の人も QnAMaker ポータルサイト へアクセスしてナレッジベースを編集することになる。 そのためには、経理部の人にも Azure アカウントを作成し、ナレッジベースの元となっている Cognitive Service のリソースへのアクセス許可を出す必要がある。

参考：

  • Collaborate with others - QnA Maker - Azure Cognitive Services | Microsoft Docs
  • Manage QnA Maker App - QnA Maker - Azure Cognitive Services | Microsoft Docs
