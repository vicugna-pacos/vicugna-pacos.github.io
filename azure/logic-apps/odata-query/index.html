title: “OData クエリ” date: 2022-07-28T16:57:23+09:00 draft: true

概要

Power Automate のいくつかのアクションで使えるクエリ言語。

Outlook

「複数のメールの取得 (V3)」アクション (操作 ID:GetEmailsV3)

参照： Microsoft Graph の $search クエリ パラメーターを使用する - Microsoft Graph | Microsoft Docs

既定の検索プロパティがある。from, subject, body の3つ。なので、単純に pizza とした場合は、from, subject, body のいずれかに “pizza” が含まれるものを検索する。 明示的に body を指定する場合は、body:pizza と書く。なお、これで pizza が本文に含まれるメッセージを検索できる。
